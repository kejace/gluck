1 Plane curves and signed curvature

This chapter sets up the elementary differential geometry of plane curves used throughout the project. We model a plane curve as a map \(\gamma : \mathbb {R} \to \mathbb {C}\), identifying the Euclidean plane \(\mathbb {R}^2\) with the complex plane \(\mathbb {C}\); this matches the complex notation \(e^{i\theta }\) used in the source for the reconstruction integral.

[Regular curve] A map \(\gamma : \mathbb {R} \to \mathbb {C}\) is a regular curve if it is twice continuously differentiable (\(C^2\), encoded ContDiff ℝ 2) and its velocity never vanishes: \(\gamma '(t) \neq 0\) for all \(t\).

  • Gluck.signedCurvature

[Signed curvature] Source: DeTurck–Gluck, §4 (defining relations \(d\alpha /ds = (\cos \theta ,\sin \theta )\), \(d\theta /ds = \kappa \)). The signed curvature of \(\gamma : \mathbb {R} \to \mathbb {C}\) at \(t\) is

\[ \kappa _\gamma (t) \; =\; \frac{\operatorname {Re}\gamma '(t)\, \operatorname {Im}\gamma ''(t) - \operatorname {Im}\gamma '(t)\, \operatorname {Re}\gamma ''(t)}{\lvert \gamma '(t)\rvert ^{3}} . \]

This is the standard parametrization-invariant formula; for a curve parametrized by its angle of inclination \(\theta \) (so \(\gamma '(\theta ) = e^{i\theta }/\kappa (\theta )\)) it reduces to \(\kappa _\gamma (\theta ) = \kappa (\theta )\), recovering the source’s defining relation \(d\theta /ds = \kappa \).

This formula is a \(C^2\) notion: it reads off \(\gamma ''\), so it is only meaningful when \(\gamma \in C^2\) (definition 1 bakes in ContDiff ℝ 2). The realized curve of a merely continuous curvature function is only \(C^1\), so the final theorem expresses curvature realization through the intrinsic notion definition 3 below, not through this formula.

  • Gluck.RealizesCurvature

[Intrinsic curvature realization] A curve \(\gamma : \mathbb {R} \to \mathbb {C}\) realizes the curvature function \(\kappa : \mathbb {R} \to \mathbb {R}\) if \(\gamma \) is \(C^1\), regular (\(\gamma '(t) \neq 0\) for all \(t\)), and there is a differentiable tangent-angle function \(\varphi : \mathbb {R} \to \mathbb {R}\) such that, for all \(t\),

\[ \gamma '(t) \; =\; \lVert \gamma '(t)\rVert \, e^{i\varphi (t)}, \qquad \varphi '(t) \; =\; \kappa (t)\, \lVert \gamma '(t)\rVert . \]

The first equation says \(\varphi (t)\) is the angle of inclination of the unit tangent; the second is the source’s defining relation \(d\theta /ds = \kappa \) rewritten via the chain rule with arc length \(s\), \(ds/dt = \lVert \gamma '(t) \rVert \): \(\varphi '(t) = (d\varphi /ds)(ds/dt) = \kappa (t)\lVert \gamma '(t)\rVert \).

Why this is the right notion. It needs only \(\gamma \in C^1\) (no \(\gamma ''\)): the angle \(\varphi \) — not \(\gamma \) — is required to be differentiable, and for the reconstruction curve \(\alpha _\rho \) (whose velocity is \(e^{i\theta }\rho (\theta )\)) the tangent angle is exactly the parameter, \(\varphi (\theta ) = \theta \), which is smooth even when \(\alpha _\rho \) is only \(C^1\). Thus \(\alpha _\rho \) realizes a merely-continuous \(\kappa \). When \(\kappa {\gt} 0\), \(\varphi ' = \kappa \lVert \gamma '\rVert {\gt} 0\), so the tangent angle turns strictly monotonically: this is the intrinsic form of convexity (rotation index \(1\) over a period), replacing the \(C^2\) predicate definition 7 in the final theorem.

  • Gluck.signedCurvature_of_realizesCurvature

[Intrinsic realization agrees with the formula on \(C^2\) curves] If \(\gamma \) is \(C^2\) and regular and realizes \(\kappa \) (definition 3), then its signed curvature equals \(\kappa \): \(\kappa _\gamma (t) = \kappa (t)\) for all \(t\). (Bridge between the intrinsic notion and the deriv²-formula; lets the deriv²-formula and the intrinsic realization be used interchangeably on \(C^2\) data.)

Proof ▶

Write \(v(t) = \lVert \gamma '(t)\rVert \), so \(\gamma '(t) = v(t)e^{i\varphi (t)}\) with \(\varphi \) the tangent angle. Since \(\gamma \in C^2\) and \(\gamma '\) is nonvanishing, \(v\) is \(C^1\) and \(\gamma ''(t) = v'(t)e^{i\varphi (t)} + v(t) i\varphi '(t)e^{i\varphi (t)}\). Decompose into real/imaginary parts: \(\Re \gamma ' = v\cos \varphi \), \(\Im \gamma ' = v\sin \varphi \), and \(\Re \gamma '' = v'\cos \varphi - v\varphi '\sin \varphi \), \(\Im \gamma '' = v'\sin \varphi + v\varphi '\cos \varphi \). The signed-curvature numerator is

\[ \Re \gamma '\, \Im \gamma '' - \Im \gamma '\, \Re \gamma '' = v^2\varphi '(\cos ^2\varphi + \sin ^2\varphi ) = v^2\varphi ', \]

(the \(v'\) terms cancel), while \(\lVert \gamma '\rVert ^3 = v^3\). Hence \(\kappa _\gamma = v^2\varphi '/v^3 = \varphi '/v = \kappa \) by the realization relation \(\varphi ' = \kappa v\).

[Closed curve] A curve \(\gamma : \mathbb {R} \to \mathbb {C}\) is closed if it is \(2\pi \)-periodic: \(\gamma (t + 2\pi ) = \gamma (t)\) for all \(t\).

  • Gluck.IsSimpleClosed

[Simple closed curve] A curve \(\gamma \) is a simple closed curve if it is closed and injective on a fundamental period, i.e. \(\gamma \) restricted to \([0, 2\pi )\) is injective. Such a \(\gamma \) is the embedding of the circle \(S^1 = \mathbb {R}/2\pi \mathbb {Z}\) that the converse theorem produces.

  • Gluck.IsConvexCurve

[Convex curve] A regular curve \(\gamma \) is convex if its signed curvature is strictly positive everywhere: \(\kappa _\gamma (t) {\gt} 0\) for all \(t\). For a simple closed curve this is equivalent to convexity of the enclosed region.
